# AzUSEcDT

Example of the use of PyQt5 with Matplotlib

## Requirements

* Python            3.6.5
* PyQt5             5.11.3
* matplotlib        2.2.2
* pandas            0.25.0

## How to use

1. Load the files from the file test folder 
    1. First load gdp.csv
    2. Second load unemployment.csv
2. Click on plot
